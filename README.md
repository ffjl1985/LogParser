Tool for detecting and parsing a standard log file, according to it's format.

Supported types:

 # access_log
 # combined_log
 # combined_wcookie_log
 # error_log
 # linux_secure
 # db2_diag
 # postfixsyslog
 # linux_message_sys_log
 # sendmail_syslog
 # weblogic_stdout
 # websphere_activity
 # exim_main
 # exim_reject
 # mysqld
 # ttc
 # linux_bootlog
 # iis
 # w3c_extended
